# **Divant** - Divide and Transfer

As the name implies (inspired by the term Divide and Conquer), **Divant** is a web-based utility that helps people split bills easily.

Stay tuned to the deployment of this project once it's feature complete!

## Features

Divant is designed to be a lightweight, mobile-friendly bill-splitting solution:

- **Loginless Group Sessions:** No signups required. Create temporary group "pools" and share them with others via a unique link or QR code.
- **Dynamic Rejoining:** Securely rejoin your active pool sessions automatically via browser cookies or by entering your previously used name.
- **Flexible Splitting Modes:** Split bills equally, with custom percentages/shares, or by fixed specific amounts.
- **Many-to-Many Balance Tracking:** View clear summaries of who owes whom and who is owed within the group.
- **Payment Status & Tracking:** Settle outstanding balances directly, track payment progress on bills, and toggle status easily.
- **Mutual Debt Offsets:** Instantly cancel out reciprocal debts between group members to simplify payments.

## Getting Started

Run the development server with `npm`:

```bash
npm run dev:ws
```

The main address of the local server will be at [http://localhost:3000](http://localhost:3000)
